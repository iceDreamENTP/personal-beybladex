# 術語

## 狀態

- `owned`：已持有，現階段可實際使用。
- `ordered`：已訂購但尚未收到，不得假設版本或實際零件。
- `wishlist`：想買但尚未訂購。

## 四層配置代號

| 代號 | 意義 | 定義 |
|---|---|---|
| `org` | Original | Takara Tomy 原廠配置與官方設計策略 |
| `opt` | Optimal | 純粹依物理結構推導的單顆理論最佳配置 |
| `dck` | Deck | 核心三機的目標永久配置 |
| `tmp` | Temporary | 依目前已實際擁有的零件可組出的暫時配置 |

## 零件

- `Blade`：上層攻擊環或主體零件。
- `Ratchet`：中層棘輪零件。
- `Bit`：底部軸心零件。

## 完成度

- `⭐☆☆☆☆`：僅有基本資料或仍待到貨、待實測。
- `⭐⭐⭐⭐☆`：已經有明確實測支持，目前接近完成。

## 隨機款

`random_variant: true` 代表已知是隨機款。尚未收到或尚未拆封確認前，只能記錄主打款資訊，不得推測實際版本或視為已取得零件。

## unknown

`unknown` 代表未確認資訊。未知欄位必須維持 `unknown`，不可用推測補齊。
